# Setup Guide

Follow these steps to set up the workspace for development.

## Prerequisites
- Docker & Docker Compose
- Python 3.11+
- Node.js 18+

## Quick Start
1. Run `docker-compose up -d` in the `docker` directory to boot dependencies.
2. Setup backend environment and run `pip install -r backend/requirements.txt`.
3. Setup frontend environment, run `npm install` and `npm run dev` in `frontend` directory.
