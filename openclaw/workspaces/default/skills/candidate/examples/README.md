# Examples

- **Search:** “Find Priya Shah by email.” Search candidates, return exact and ambiguous results, then show linked applications.
- **Create:** “Add the attached applicant.” Review duplicate candidates first; after confirmation, create only verified profile fields and report the new id.
- **Merge review:** “These two records may be the same person.” Produce a side-by-side comparison and request approval; do not merge.
- **Summary:** “Summarize this candidate.” Read the profile and related recruiting records, then label facts and interpretation separately.
