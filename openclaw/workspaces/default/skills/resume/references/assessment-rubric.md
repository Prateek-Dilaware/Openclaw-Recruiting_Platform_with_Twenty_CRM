# Resume Assessment Rubric

Use only role-relevant criteria available in the requisition or explicitly supplied by a recruiter:

1. Required skills and demonstrated evidence.
2. Relevant scope, domain, and responsibilities.
3. Experience level against stated requirements.
4. Outcomes, impact, and technical depth where supported by the resume.
5. Explicit gaps, ambiguities, and interview follow-up questions.

Do not use protected characteristics, names, photos, addresses, graduation dates, or other irrelevant personal attributes as scoring evidence. A score is an explainable decision aid, not an automated hiring outcome.
