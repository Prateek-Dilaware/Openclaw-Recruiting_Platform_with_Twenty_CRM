# Communication Standard

## Draft rules

- State only verified, authorized facts.
- Separate internal recruiter rationale from external candidate wording.
- Do not promise outcomes, compensation, start dates, accommodations, or legal terms unless explicitly approved.
- For rejection and feedback, avoid unsupported performance claims and discriminatory language.
- For offers, label all unresolved terms and require authorized offer status before external use.

## Delivery boundary

No skill sends email. A future Gmail/Outlook MCP can deliver an already approved draft; this skill must not simulate delivery.
